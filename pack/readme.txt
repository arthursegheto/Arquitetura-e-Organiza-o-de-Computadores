Esse projeto consiste em quebrar a senha de um arquivo.zip de 5 caracteres, sendo eles caracteres da tabela ASCII da posição 33 até 126. Os códigos utilizam de programação paralela, utilizando multithreading e forçando todos os núcleos e threads do processador trabalharem em conjunto de forma que descubram a senha do arquivo. Após a senha ser descoberta, a execução do programa encerra e o arquivo.zip é extraído.

English translation:

This project consists of breaking the password of a 5-character .zip file, which are characters from the ASCII table from position 33 to 126. The codes use parallel programming, using multithreading and forcing all processor cores and threads to work together so that they discover the password of the file. After the password is guessed, program execution ends and the .zip file is extracted.

Traduction Française:

Ce projet consiste à casser le mot de passe d'un fichier .zip à 5 caractères, qui sont des caractères de la table ASCII de la position 33 à 126. Les codes utilisent la programmation parallèle, en utilisant le multithreading et en forçant tous les cœurs et threads du processeur à travailler ensemble afin qu'ils découvrir le mot de passe du fichier.
