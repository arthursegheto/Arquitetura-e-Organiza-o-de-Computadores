Português:

Este projeto também utiliza de programação paralela, dividindo as tarefas em threads e utilizando de todos os núcleos do processador.
O objetivo do trabalho era implementar um algoritmo que procurasse entre diretórios em todos os arquivos .txt presentes e retornasse ao fim
o maior número primo e o caminho do mesmo em que se encontrava. A tarefa foi concluída com sucesso.

English:

This project also uses parallel programming, dividing tasks into threads and using all processor cores.
The objective of the work was to implement an algorithm that searches among directories in all .txt files present and returns at the end
the largest prime number and the path it was on. The task completed successfully.

Français:
